# Medicus---Trusted-Drug-and-Medicine-Guide
Medicus is a responsive healthcare web application designed to provide reliable medicine and drug information for patients and healthcare professionals. The platform offers symptom-based medicine search, voice search functionality, secure authentication, and an interactive A–Z medicine directory with a simple and user-friendly interface.

Features
Symptom-based medicine search
Voice search using microphone input
A–Z medicine directory
Secure Login/Signup system
Responsive and interactive UI
Technologies Used
HTML
CSS
JavaScript
React
Firebase/MySQL
Objective

To simplify access to medicine-related information through an efficient and user-friendly digital healthcare platform.
